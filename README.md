# Caesar Cipher Encryption and Decryption

## Project Overview
This project implements the Caesar Cipher algorithm using Python. The Caesar Cipher is one of the oldest and simplest encryption techniques in cryptography. It works by shifting each letter in a message by a fixed number of positions in the alphabet.

## Objective
To develop a Python program that can encrypt and decrypt text messages using a user-defined shift value.

## Features
- Encrypt plain text messages
- Decrypt encrypted messages
- User-defined shift value
- Supports uppercase and lowercase letters
- Preserves spaces and special characters

## Technologies Used
- Python

## How the Program Works
1. The user enters a message.
2. The user enters a shift value.
3. The program encrypts the message using the Caesar Cipher algorithm.
4. The encrypted message is displayed.
5. The program then decrypts the encrypted message back to the original text.

## How to Run the Project

1. Install Python.
2. Download or clone this repository.
3. Open the project folder in VS Code.
4. Open the terminal.
5. Run the following command:

```bash
python caesar_cipher.py
```

6. Enter the message and shift value when prompted.

## Example

Input:

Message: HELLO

Shift Value: 3

Output:

Encrypted Text: KHOOR

Decrypted Text: HELLO
